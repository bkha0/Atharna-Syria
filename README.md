# آثارنا - توثيق القلاع والمواقع الأثرية في سوريا

## الرؤية
موقع تفاعلي يعرض القلاع السورية وتاريخها وموقعها الجغرافي، باستخدام خرائط Leaflet وقائمة ديناميكية.

## القائمة الأولية للمواقع
- قلعة دمشق
- قلعة حلب
- قلعة الحصن (حمص)
- قلعة صلاح الدين الأيوبي (اللاذقية)
- قلعة حماة
- قلعة الرقة
- قلعة الرحبة (دير الزور)
- قلعة سكرة (الحسكة)
- بصرى الشام (درعا)

## التقنيات المستخدمة
- HTML5 / CSS3
- JavaScript (ES6)
- Leaflet.js للخرائط
- Git & GitHub
