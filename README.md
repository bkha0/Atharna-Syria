# آثارنا - توثيق القلاع والمواقع الأثرية في سوريا

## الرؤية
موقع تفاعلي يعرض القلاع السورية وتاريخها وموقعها الجغرافي، باستخدام خرائط وقائمة ديناميكية.

## القائمة الأولية للمواقع
- قلعة دمشق
- قلعة حلب
- قلعة الحصن (حمص)
- قلعة حماة
- قلعة الرقة
- قلعة الرحبة (دير الزور)
- قلعة سكرة (الحسكة)
- بصرى الشام (درعا)

## التقنيات المستخدمة
- HTML5 / CSS3
- JavaScript (ES6)
- Leaflet.js
- Git & GitHub
